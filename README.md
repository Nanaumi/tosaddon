# [MARKET GOTO PAGE]

SYNOPSIS: ``/goto [<pagenumber>|last|first]``
          
DESCRIPTION: Adds a command to go through market pages faster.

EXAMPLES: 
- ``/goto 50``
- ``/goto last``
- ``/goto first``
